# Tukan JSFX Plugins
Main repository for TukanStudio's JSFX plugins for Reaper.

## Information and Usage
John has a Patreon you can support him at! [Tukan Studtio's Patreon](https://www.patreon.com/tukanstudios)

Be sure to subscribe to John's YouTube for plugin demos and plugin explanations. [YouTube](https://www.youtube.com/channel/UCoU5ZhDHHAm6KilILToHRvg)

John has not specified what software license these plugins fall under. John, please update this line or specify in Github.

You can add these plugins to Reaper via ReaPack by following this [link for instructions.](https://stash.reaper.fm/v/43504/TUKANPLUGINS.png)
